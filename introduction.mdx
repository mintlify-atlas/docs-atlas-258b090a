## Who uses EspoCRM

- **Sales teams** — Track leads through a pipeline, manage opportunities, log calls and meetings, and forecast revenue with built-in Kanban boards
- **Support teams** — Manage customer cases, link them to contacts and accounts, and use the knowledge base for self-service
- **Marketing teams** — Run campaigns, segment target lists, send mass emails, and track campaign performance
- **Developers** — Extend EspoCRM with custom entities, hooks, formula functions, and full-featured extensions

## Key highlights

<AccordionGroup>
  <Accordion title="Open source and self-hosted">
    EspoCRM is licensed under AGPL-3.0, meaning the full source code is publicly available. You host it on your own server, giving you complete control over your data and infrastructure. No vendor lock-in, no per-user fees beyond your hosting costs.
  </Accordion>
  <Accordion title="Built-in customization without code">
    The Entity Manager lets you create new record types, add custom fields of 40+ types, define relationships, and configure layouts — all through the admin UI. No code required for most customizations.
  </Accordion>
  <Accordion title="Powerful formula scripting">
    EspoCRM's formula language lets you define calculated fields, automate field updates on save, validate data, and trigger side effects. Formulas run server-side and can access related record data.
  </Accordion>
  <Accordion title="Full REST API with OpenAPI">
    Every entity in EspoCRM is automatically accessible through the REST API. The system generates an OpenAPI-compatible schema that reflects your custom entities and fields. Authentication supports API keys, HMAC signatures, and bearer tokens.
  </Accordion>
</AccordionGroup>

## Get started

<Steps>
  <Step title="Install EspoCRM">
    Deploy on your server using the manual installer, a shell script, or Docker. See the [Installation guide](/installation) for all options.
  </Step>
  <Step title="Configure your instance">
    Set up email, define users and roles, and customize your entities through the Admin panel. The [Administration section](/administration/users-and-teams) walks you through the key steps.
  </Step>
  <Step title="Import your data">
    Use the built-in CSV import to bring in existing contacts, accounts, and leads. Map columns to fields and run the import from **Admin → Import**.
  </Step>
  <Step title="Integrate via API">
    Connect EspoCRM to other tools using the REST API. See [API Overview](/api/overview) and [Authentication](/api/authentication) to get started.
  </Step>
</Steps>
